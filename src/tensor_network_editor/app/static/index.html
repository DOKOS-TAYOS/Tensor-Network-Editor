How to use the editor

A quick guide for navigation, selection, copy and paste, groups, the Contract planner, exports, and shortcuts.

Navigation

  • Use the mouse wheel or trackpad to zoom.
  • Drag empty canvas space to move the view.
  • Use the minimap to recenter or drag the visible area.

Selection

  • Left click selects one tensor, index, or connection.
  • Drag with right click to select many elements at once.
  • Use Ctrl/Cmd+C and Ctrl/Cmd+V to copy and paste a tensor subgraph.

Organize and edit

  • Drag one selected tensor to move all selected tensors together.
  • Use the multi-selection panel to recolor or delete everything selected.
  • If only tensors are selected, you can add one default index to each tensor.
  • Resize tensors from the corner handles directly on the canvas.
  • Use Create Group to keep related tensors together and drag the group box as one block.
  • Notes can be moved, resized, collapsed, and expanded directly on the canvas.
  • Notes, groups, colors, and copied selections are all part of the saved design.

Connect and contract

  • Activate Connect and click two open indices with the same dimension.
  • Activate Contract in the planner, then click tensors or previous steps to build a manual contraction path.
  • Automatic global and local suggestions can be previewed visually before you accept them.
  • Export PNG for a clean image or SVG for a vector version.
  • Undo and redo affect the design, not the current zoom or viewport.
