How to use the editor

A quick guide for navigation, selection, connect mode, exports, and shortcuts.

Navigation

  • Use the mouse wheel or trackpad to zoom.
  • Drag empty canvas space to move the view.
  • Use the minimap to recenter or drag the visible area.

Selection

  • Left click selects one tensor, index, or connection.
  • Drag with right click to select many elements at once.

Batch actions

  • Drag one selected tensor to move all selected tensors together.
  • Use the multi-selection panel to recolor or delete everything selected.
  • If only tensors are selected, you can add one default index to each tensor.

Connect and export

  • Activate Connect and click two open indices with the same dimension.
  • Export PNG for a clean image or SVG for a vector version.
  • Undo and redo affect the design, not the current zoom or viewport.
